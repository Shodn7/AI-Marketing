# 🚀 AI-Marketing Platform

An AI-powered marketing platform developed during the WeCloudData × Saudi Digital Academy Hackathon.

The platform helps merchants generate high-quality marketing content with minimal effort by leveraging Generative AI.

## 🌟 Features

- 🛍️ Product management dashboard
- 🎨 AI-generated marketing images
- ✍️ One-click commercial post generation
- 👀 Approval workflow before publishing
- 📱 User-friendly merchant interface
- 🤖 AI-powered content assistance

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Shodn7/AI-Marketing.git
cd AI-Marketing/HackathonSDE
```

### Backend Setup

```bash
cd backend

# Activate virtual environment
# Windows
.venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run server
uvicorn main:app --reload
```

Backend runs on:

```text
http://127.0.0.1:8000
```

### Frontend Setup

```bash
cd frontend

npm install
npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```
